# ts-koa
这是一个空环境

### 项目结构

```
koa-ts
├─.env                      // 环境配置
├─README.md                 //
├─app.ts                    // 入口文件
├─config.ts                 // 环境文件导入
├─nodemon.json              // nodemon配置文件
├─package-lock.json
├─package.json
├─tsconfig.json
├─test                      // 可以写测试 做其他测试
├─src                       
|  ├─router.ts              // 路由文件
|  ├─controller             // 控制器
|  |     ├─BaseController.ts
|  |     └DefaultController.ts                  
├─runtime                   // 缓存
├─logs                      // 日志
```
